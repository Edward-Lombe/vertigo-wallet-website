Vertigo - The Lightweight Vertcoin Wallet

Vertigo is a Lightweight Vertcoin Wallet designed to end the frustrating waiting time downloading gigabytes of block chain! Vertigo has an extremely smooth interface. It was created with the intent and success of being exceedingly end-user friendly. Vertigo utilizes an interface that is uncomplicated and easy to navigate. New features are added upon request!

Downloads

Version 0.3.9.1 - Source code available on Github

Features

  • Send Vertcoins
  • Receive Vertcoins
  • Request Payments
  • Ultra Fast Synchronization
  • Saves Hard Disk Space
  • Paper Wallet Creation & Print
  • Paper Backup of Wallet Private Keys
  • Import Paper Wallet Private Key
  • QR Code Support: Vertcoin Address, Vertcoin Payment Request, Vertcoin Private Key with Webcam Scanning, Drag & Drop, Copy & Paste
  • Wallet File Storage in User Specified Location
  • Send All Vertcoins
  • Label Address
  • See Address Balance
  • Export/Import Private Keys: All Known Formats
  • Encrypt/Decrypt Wallet with Password to Protect Private Keys
  • Transaction History
  • Addressbook
  • Wallet Backup Notifications to Protect Coins
